Tienda

Este proyecto es una aplicación en lenguaje C diseñada para gestionar el inventario de una tienda. Permite registrar productos, organizarlos por categorías y calcular información relevante del stock.

Características principales

Registro de productos con:

- Nombre

- Precio

- Stock

- Categoría (alimentos, ropa, electrónica)

- Carga dinámica de información según la categoría (usando union para atributos específicos).

- Visualización completa del inventario.

- Cálculo del valor total del stock.

- Estructura modular con funciones separadas.

- Implementación de struct y union para optimizar memoria.

Estructura del proyecto

- Uso de estructuras para modelar productos.

- Menú interactivo para navegar entre opciones.

- Validaciones básicas de entrada.

- No utiliza bases de datos ni archivos externos (todo en memoria en tiempo de ejecución).

Objetivo

- Aprender manejo de estructuras, uniones, arrays, y lógica aplicada a un sistema realista de inventario.
